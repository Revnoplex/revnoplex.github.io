Data Collection Policy:

Revnobot will collect and store certain types of data for developer/debugging and for detecting potential misuse. The types of data collected can be listed below

  • Server Names and IDs
  • Server Owner Usernames and IDs
  • Invite Access to servers if granted invite permission
  • Messages: Revnobot will only collect messages related to command execution. What is collected:
    • Messages that start with the bot's prefix
    • Messages that the bot sent.
    • Slash, user and message commands executed to the bot.
    • The name of the command used and the arguments passed to the bot
    • DMs sent to the bot

Terms of Use:

  1. Do not abuse or spam the bot, there is a cooldown to prevent that from happening but this is in place just in case the cooldown doesnt work.
  2. Do not try to exploit the bot. if you find a problem with the bot, please tell me immidiately (dm me on social media).

Actions:

If the bot is being exploited or spammed. The offending server can be blocklisted.
