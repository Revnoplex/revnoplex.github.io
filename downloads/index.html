Gamecube System menu

    
Instructions: In Dolphin Emulator under /GC Paste the corresponding IPL.bin into the region folder based on its region/Video Technology. 
Example: if you downloaded the PAL version, Paste it in the /GC/EUR folder. 
In dolphin versions 5.0 to 5.0-3899 Under the configuration menu and under the "GameCube" tab, make sure "Skip Main Menu" is unchecked to make gamecube game boot through the menu upon startup. 
In dolphin 5.0-3902 or later the option "Skip BIOS" is called "Skip Main Menu" and the menu can also be manually be run under Tools, Load GameCube Main Menu. 
see more at GameCube Main Menu - Dolphin Emulator Wiki
