Data Collection Policy

Revnobot will collect and store certain types of data for developer/debugging and for detecting potential misuse.

This data is only stored locally on the bot's host machine.

The types of data collected include the following:

  • Server Names and IDs
  • Server Owner Usernames and IDs
  • Invite Access to servers if granted invite permission
  • Messages: Revnobot will only collect messages related to command execution. What is collected:
    • Messages that start with the bot's prefix
    • Messages that the bot sent.
    • Slash, user and message commands executed to the bot.
    • The names of the command passed to the bot
    • DMs sent to the bot
